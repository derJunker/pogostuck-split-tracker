Hi! Ty for using my Program :)

Before you can use the splits-program, you need to set up a few things.

  • Add Steam Start-Option: You need to add "-diag" as a start-option to your Pogostuck steam execution config. This is needed so the game writes the logs that this program reads for split detection etc.
    To do this, right-click on Pogostuck in your Steam Library, go to Properties, then Launch Options and add "-diag" (without quotes) to the text field.
  • Steam Userdata Path: You need to input your steam userdata path (in the Paths tab). It is used to read through your Settings.txt, so i can read out your current split times.
    Most likely it will be something similar to this:
    C:\Program Files (x86)\Steam\userdata\$steam-user-id\688130\remote
  • Pogo Path: You need to input your Pogostuck path (in the Path tab as well). It is needed for basically everything. In there the before mentioned logs are put.
    It should be looking similar to this:
    C:\SteamLibrary\steamapps\common\Pogostuck
  • Enter Pbs: Currently i dont have a way to automatically detect your Pbs. So you have to enter it manually. So you need to do that under the TODO tab.
  • Enter Split Skips: If you do any skips that skip a whole split - notably palm- and anvil-skip - you need to set this manually. Otherwise the sum-of-best-splits calculation will be wrong.
    You can do this under the Split-Skips tab.
